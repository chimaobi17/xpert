# Superpowers MCP Plugin

Installed and configured for advanced coding superpowers.

## What Is It?

Superpowers-MCP provides advanced skills for:
- **TDD (Test-Driven Development)** — Structured test-first development
- **Debugging** — Advanced debugging workflows and strategies
- **Collaboration** — Best practices for team workflows
- **Best Practices** — Code quality and architectural patterns

## Installation

✅ **Status**: Installed

```bash
Package: superpowers-mcp
Location: ~/.npm-global/node_modules/superpowers-mcp
Version: Latest
```

## Configuration

Registered in `.mcp-servers.json`:

```json
{
  "superpowers": {
    "command": "node",
    "args": ["/Users/chimzy/.npm-global/node_modules/superpowers-mcp/out/server.js"],
    "stdio": true
  }
}
```
